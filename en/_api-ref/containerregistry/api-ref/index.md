# Yandex Container Registry API
This API reference is organized by resource. All actions are performed by sending HTTP requests to resource URLs. For more information about Yandex.Cloud API architecture, see [API Concepts](/docs/api-design-guide/).

Resource | Description
--- | ---
[Image](Image/index.md) | A set of methods for managing Image resources.
[Registry](Registry/index.md) | A set of methods for managing Registry resources.
[Repository](Repository/index.md) | A set of methods for managing Repository resources.
