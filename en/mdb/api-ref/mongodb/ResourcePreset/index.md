# ResourcePreset
A set of methods for managing ResourcePreset resources.
## JSON Representation {#representation}
```json 
 {
  "id": "string",
  "zoneIds": [
    "string"
  ],
  "cores": "string",
  "memory": "string"
}
```

## Methods {#methods}
Method | Description
--- | ---
[get](get.md) | Returns the specified ResourcePreset resource.
