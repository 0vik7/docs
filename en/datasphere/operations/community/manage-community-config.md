# Managing configurations in community projects

{% note info %}

You can select computing resource [configurations](../../concepts/configurations.md) to be available across community projects only if you have the `{{ roles-datasphere-communities-admin }}` role in the community.

{% endnote %}

1. Open the {{ ml-platform-name }} [home page]({{ link-datasphere-main }}).
1. In the left-hand panel, select ![community-panel](../../../_assets/datasphere/communities.svg) **{{ ui-key.yc-ui-datasphere.common.spaces }}**.
1. Select the community to change the computing resource configurations in.
1. Click the **{{ ui-key.yc-ui-datasphere.common.restrictions }}** tab.
1. Select the available configurations in the **{{ ui-key.yc-ui-datasphere.spaces-page.access-in-project }}** column.
