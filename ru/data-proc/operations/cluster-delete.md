# Удаление кластера

Вы можете удалить кластер вместе со всеми хранимыми данными. Помните, что резервные копии данных не создаются автоматически: если вы хотите сохранить данные для восстановления кластера или последующего использования, используйте, например, сервис Yandex Object Storage.

{% list tabs %}

- Консоль управления

    1. Откройте страницу каталога в консоли управления.
    1. Выберите сервис **Data Proc**.
    1. Нажмите значок ![image](../../_assets/options.svg) для нужного кластера и выберите пункт **Удалить**.
    1. Подтвердите удаление кластера.

    Data Proc запустит операцию удаления кластера.
