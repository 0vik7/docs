# Поддерживаемая версия Hadoop и дополнительных компонентов

Создавая кластеры [!KEYREF dataproc-name] на этапе Превью, вы можете использовать образ Hadoop со следующим набором компонент:

* Hadoop 2.8.5
* Tez 0.9.1
* Hive 2.3.4
* ZooKeeper 3.4.6
* HBase 1.3.3
* Sqoop 1.4.6
* Oozie 4.3.0
* Spark 2.2.1
* Flume 1.8.0
* Zeppelin 0.7.3
