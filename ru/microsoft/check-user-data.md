# Требования к учетной записи {{ yandex-cloud }}

Одно из условий использования лицензионных продуктов Microsoft — предоставление дополнительных пользовательских данных. Если в вашем платежном аккаунте эти данные отстутствуют или указаны некорректно, мы будем вынуждены приостановить доступ ко всем сервисам и лицензиям Microsoft.

## Обязательные данные в платежном аккаунте {#required-data}

Для юридических лиц:

* Наименование организации.
* ИНН.
* Адрес.
* Страна.

Для физических лиц:

* Имя и фамилия.
* Адрес.
* Страна.
