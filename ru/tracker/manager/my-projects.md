# Список моих проектов

Откройте список всех доступных вам проектов:

1. На панели слева выберите ![](../../_assets/tracker/svg/project.svg)&nbsp;**Проекты** или перейдите по [прямой ссылке]({{ link-tracker }}pages/projects) и откройте страницу проекта.

1. Чтобы найти проекты, выберите фильтр:

    * **Мои проекты** для всех проектов, в которых вы являетесь автором, ответственным, участником или заказчиком.

    * **Я автор**, **Я ответственный**, **Я участник** или **Я заказчик** для проектов, в которых вы выполняете соответствующую роль.

    * **Избранные** для проектов, которые были добавлены в <q>Избранное</q>.

    * **Недавно просмотренные** для проектов, которые вы открывали последними.

1. Если отобразились не все нужные проекты, откройте полный список:

    * На панели слева в нижней части страницы нажмите кнопку **Полный список проектов**.

    * Чтобы открыть все проекты, которые доступны вам в {{ tracker-name }}, в нижней части окна нажмите кнопку **Все проекты** или перейдите по [прямой ссылке]({{ link-tracker }}pages/projects).

1. Для поиска проектов в списке используйте фильтры:

    * Чтобы найти проект по названию, введите его в поле **Поиск**.

    * Чтобы отобразить список проектов в виде календарного плана, используйте [**Диаграмму Ганта**](../gantt/list-of-projects.md). В календарном плане вы можете указать сроки работы над проектом и назначить ответственного.

    * Чтобы выбрать параметр поиска по полю проекта, нажмите на его название над списком. Для выбора дополнительных параметров нажмите кнопку ![](../../_assets/tracker/svg/add-filter.svg) **Добавить поле для фильтрации** и выберите поле.

    * Чтобы настроить порядок отображения, нажмите кнопку ![](../../_assets/tracker/svg/sorting.svg) **Сортировка** и выберите параметр.

    Настроенный фильтр сохранится и будет отображаться после обновления страницы или при повторном переходе в список проектов.

    {% note tip %}

    Чтобы поделиться ссылкой на отфильтрованный список проектов, скопируйте ссылку из адресной строки браузера.

    {% endnote %}

1. Для просмотра карточки проекта нажмите на его название. Информация о проекте отобразится на боковой панели справа, если в настройках интерфейса {{ tracker-name }} включена [функция **Открывать задачи на боковой панели**](../user/personal.md#choose-theme).
