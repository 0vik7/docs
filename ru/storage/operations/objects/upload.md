# Загрузка объекта

---

**[!TAB Консоль управления]**

Консоль управления позволяет работать с корзинами [!KEYREF objstorage-name] как с иерархической файловой системой. Пользователь может создавать внутри корзины папки и загружать туда объекты. При этом необходимо помнить, что в SDK и HTTP API ключом объекта будет весь путь к объекту от корня корзины. Читайте подробнее в разделе [[!TITLE]](../../concepts/object.md).

Чтобы загрузить объект:

1. В консоли управления выберите каталог.
1. Нажмите плитку **Yandex Object Storage**.
1. Нажмите на имя необходимой корзины.
1. Если вы хотите загрузить объект в конкретную папку, перейдите в нее, нажав на имя. Если хотите создать новую папку, создайте ее, нажав на кнопку **Создать папку**.
1. Оказавшись в нужной папке, нажмите кнопку **Загрузить**.
1. В появившемся окне выберите необходимые файлы и нажмите кнопку подтверждения.
