# Загрузка объекта

---

**[!TAB Консоль управления]**

Консоль управления позволяет работать с бакетами [!KEYREF objstorage-name] как с иерархической файловой системой. Пользователь может создавать внутри бакета папки и загружать туда объекты. При этом необходимо помнить, что в SDK и HTTP API ключом объекта будет весь путь к объекту от корня бакета. Читайте подробнее в разделе [[!TITLE]](../../concepts/object.md).

>[!NOTE]
>
>Через консоль управления нельзя загрузить объекты размером более 5 ГБ (см. [[!TITLE]](../../concepts/limits.md)). Для загрузки больших объектов используйте другие [инструменты](../../instruments/index.md).

Чтобы загрузить объект:

1. В консоли управления выберите каталог.
1. Нажмите плитку **Object Storage**.
1. Нажмите на имя необходимого бакета.
1. Если вы хотите загрузить объект в конкретную папку, перейдите в нее, нажав на имя. Если хотите создать новую папку, создайте ее, нажав кнопку **Создать папку**.
1. Оказавшись в нужной папке, нажмите кнопку **Загрузить**.
1. В появившемся окне выберите необходимые файлы и нажмите кнопку **Открыть**.
1. Консоль управления отобразит все объекты, выбранные для загрузки и предложит для каждого из них выбрать [класс хранилища](../../concepts/storage-class.md). По умолчанию установлено Стандартное хранилище.
1. Нажмите кнопку **Загрузить**.
