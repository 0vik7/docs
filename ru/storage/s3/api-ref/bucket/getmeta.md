# Метод getMeta

Возвращает метаданные корзины или ошибку.

Позволяет проверить:

- Существует ли корзина.
- Достаточно ли у пользователя прав для доступа к корзине.



## Запрос {#request}

```
HEAD /{bucket} HTTP/1.1
```

### Path параметры {#path-parameters}

Параметр | Описание
----- | -----
`bucket` | Имя корзины.


### Заголовки {#request-headers}
Используйте в запросе только [общие заголовки](../common-request-headers.md).

## Ответ {#response}

### Заголовки {#response-headers}

Ответ может содержать только [общие заголовки](../common-response-headers.md).

### Коды ответов {#response-codes}

Перечень возможных ответов смотрите в разделе [[!TITLE]](../response-codes.md).

Успешный ответ не содержит дополнительных данных и означает, что корзина существует и у пользователя достаточно прав для доступа к ней.
