# Общие заголовки ответов

Помимо [стандартных HTTP-заголовков](https://en.wikipedia.org/wiki/List_of_HTTP_header_fields) Yandex [!KEYREF objstorage-name] использует дополнительные заголовки, которые могут присутствовать в ответах.

В таблице ниже описаны дополнительные заголовки, а также стандартные заголовки, к которым необходимы дополнительные пояснения.

Заголовок | Описание
----- | -----
`ETag` | Хэш объекта.<br/><br/>Не включает метаданные объекта.
`x-amz-id-2` | Специальный идентификатор, который используется вместе с заголовком `x-amz-request-id` при решение проблем с [!KEYREF objstorage-name].
`x-amz-request-id` | Уникальный идентификатор запроса.<br/><br/>Может понадобиться при обращении в поддержку Яндекс.Облака при возникновении проблем.
