# Удалить рабочий стол

{% list tabs %}

- Консоль управления

  1. В [консоли управления]({{ link-console-main }}) выберите каталог, в котором расположен рабочий стол.
  1. В списке сервисов выберите **{{ cloud-desktop-name }}**.
  1. На панели слева выберите ![image](../../../_assets/cloud-desktop/desktops.svg) **Рабочие столы**.
  1. Справа от рабочего стола, который вы хотите удалить, нажмите ![image](../../../_assets/options.svg) и выберите **Удалить**.

      {% note tip %}

      Чтобы удалить несколько рабочих столов, выделите их в списке и нажмите кнопку **Удалить** в нижней части экрана.

      {% endnote %}

  1. Подтвердите удаление.

{% endlist %}
