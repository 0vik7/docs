# Поддерживаемые клиенты СУБД

Managed Service for ClickHouse гарантирует работу только для определенных клиентов СУБД, но не накладывает ограничений на использованиие остальных клиентов.

Поддерживаются следующие клиенты:

- `clickhouse-client`;
- клиенты, которые используют ClickHouse HTTP API и поддерживают SSL.
