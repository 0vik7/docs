# Backup
Набор методов для управления ресурсами Backup для PostgreSQL.
## JSON-представление
```json 
 {
  "id": "string",
  "folderId": "string",
  "createdAt": "string",
  "sourceClusterId": "string",
  "startedAt": "string"
}
```

## Методы
Метод | Описание
--- | ---
[get](get.md) | Возвращает указанный ресурс Backup для PostgreSQL.
[list](list.md) | Получение списка ресурсов Backup для указанного каталога.
