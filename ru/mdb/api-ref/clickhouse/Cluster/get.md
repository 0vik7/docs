# Метод get
Возвращает указанный ресурс Cluster для ClickHouse.
 
Чтобы получить список доступных ресурсов Cluster для ClickHouse,
отправьте запрос [list](/docs/mdb/api-ref/clickhouse/Cluster/list).
 
## HTTP-запрос
`GET /managed-clickhouse/v1/clusters/{clusterId}`
 
## Path-параметры {#path_params}
 
Name | Description
--- | ---
clusterId | Обязательное поле. Идентификатор возвращаемого ресурса Cluster для ClickHouse. Чтобы получить идентификатор кластера, используйте запрос [list](/docs/mdb/api-ref/clickhouse/Cluster/list).  Максимальная длина — 50 символов.
 
## Ответ {#responses}
**HTTP Code: 200 - OK**

Ресурс Cluster для ClickHouse. Подробнее см. раздел [Взаимосвязь ресурсов сервиса](/docs/mdb/concepts).
