# Сервисы Яндекс.Облака

_Сервис_ в составе Яндекс.Облака позволяюет управлять определенными категориями облачных ресурсов и доступом к ним.
На данный момент в рамках платформы Яндекс.Облако доступны следующие сервисы:

  - [Yandex Compute Cloud](../../compute/) — сервис облачных вычислений.
  - [Yandex Object Storage](../../storage/) — сервис объектного хранилища.
  - [Yandex Identity and Access Management](../../iam/) — сервис по управлению доступом к облачным ресурсам.
  - [Yandex Resource Manager](../../resource-manager/) — сервис по управлению облачными ресурсами.
  - [Yandex Virtual Private Cloud](../../vpc/) — сервис для управлениями сетями.
  - [Yandex Managed Databases](../../mdb/) — сервис управления данными.
  - [Yandex SpeechKit](../../speechkit/) — сервис речевых технологий.
  - [Yandex Translate](../../translate/) — сервис машинного перевода.


### Как подключиться к сервисам

Для подключения к сервисам и управления ресурсами вы можете использовать консоль управления Яндекс.Облака. С ее помощью вы можете запускать виртуальные машины в Яндекс.Облаке и подключаться к ним, обращаться к объектному хранилищу, создавать базы данных и т. п.
