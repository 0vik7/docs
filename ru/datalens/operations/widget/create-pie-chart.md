# Создание круговой диаграммы

Круговая диаграмма — выборка из значений показателей относительно их общей суммы.

1. На главной странице сервиса Yandex DataLens нажмите **Создать виджет**.
1. В разделе **Датасет** выберите датасет для визуализации.
1. В разделе визуализации выберите **Круговая диаграмма**.
1. Перетащите измерение из датасета в поле **Измерения**.
1. Перетащите показатель из датасета в поле **Показатели**. Значения отобразятся в виде областей круговой диаграммы.
1. Перетащите измерение или показатель из датасета в поле **Фильтры**. Поле может быть пустым, тогда фильтрация не будет применена.
