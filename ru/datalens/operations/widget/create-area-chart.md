# Создание графика площади

График площади — диаграмма с областями и накоплением. Показывает отношение частей к целому.

Для создания графика площаи выполните следующее:
1. На главной странице сервиса Yandex DataLens нажмите **Создать виджет**.
1. В разделе **Датасет** выберите датасет для визуализации.
1. В разделе визуализации выберите **График площади**.
1. Перетащите измерение из датасета в секцию **X**. Значения отобразятся в нижней части графика по оси X.
1. Перетащите один или несколько показателей из датасета в секцию **Y**. Значения отобразятся в виде графика по оси Y в виде областей.
1. Перетащите измерение в секцию **Цвет**. Значение показателя разделится по цветам в зависимости от выбранного измерения.
1. Перетащите измерение или показатель из датасета в секцию **Фильтры**. Поле может быть пустым, тогда фильтрация не будет применена.
