# Создание таблицы среза

Таблица среза — выборка из набора данных по значению двух измерений.

Для создания таблицы среза выполните следующее:
1. На главной странице сервиса Yandex DataLens нажмите **Создать виджет**.
1. Выберите датасет для визуализации.
1. В разделе визуализации, выберите тип визуализации поле **Таблица среза**.
1. Перетащите измерение из датасета в секцию **Колонки**. 
1. Перетащите измерение из датасета в секцию **Строки**.
1. Перетащите показатель из датасета в секцию **Показатели**. Значения отобразятся в ячейках таблицы.
1. Перетащите измерение или показатель из датасета в секцию **Фильтры**. Поле может быть пустым, тогда фильтрация не будет применена.
