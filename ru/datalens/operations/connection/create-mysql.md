# Создание подключения к MySQL

Чтобы создать подключения к MySQL:
1. Перейдите на [страницу подключений](https://datalens.yandex.ru/connections).
1. Нажмите кнопку **Создать подключение**.
1. Выберите коннектор **MySQL**.
    > [!IMPORTANT]
    >
    > При подключении к внешнему источнику данных (который не является ресурсом Яндекс.Облака), необходимо предоставить доступ 
    к источнику для следующих IP-адресов сервиса [!KEYREF datalens-short-name]: 
    > - `77.88.39.88` порт `29`
    > - `93.158.158.96` порт `29`
    > - `93.158.156.80` порт `29`
    > - `77.88.39.224` порт `28`
    > - `37.9.126.112` порт `28`
    > - `37.9.65.32` порт `28`
1. Укажите параметры подключения:
    - **Имя подключения**. Задайте имя подключения, может быть произвольным.
    - **Имя хоста или IP**. Задайте путь до хоста MySQL.
    - **Порт**. Задайте порт подключения к MySQL.
    - **Имя базы данных**. Задайте имя подключаемой базы данных.
    - **Имя пользователя**. Задайте имя пользователя для подключения к MySQL.
    - **Пароль**. Задайте пароль для указанного пользователя.
1. Нажмите **Подключить**. Подключение появится в списке.
> [!NOTE]
>
> Вы можете проверить подключение к хосту перед созданием. Для этого нажмите кнопку **Проверить подключение**.
