# AVG

_Функция `AVG` также доступна как [оконная](AVG_WINDOW.md)._

#### Синтаксис {#syntax}


```
AVG( value )
```

#### Описание {#description}
Возвращает среднее для всех значений. Работает с числовыми типами данных и с типами `Дата | Дата и время`.

**Типы аргументов:**
- `value` — `Дата | Дата и время | Дробное число | Целое число`


**Возвращаемый тип**: Зависит от типов аргументов

#### Примеры {#examples}

```
AVG([Profit])
```


#### Поддержка источников данных {#data-source-support}

`Материализованный датасет`, `ClickHouse 1.1`, `Microsoft SQL Server 2017 (14.0)`, `MySQL 5.6`, `Oracle Database 12c (12.1)`, `PostgreSQL 9.3`.
