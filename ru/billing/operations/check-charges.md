# Посмотреть детализацию использования сервисов

На странице **Детализация** вы можете посмотреть подробные сведения по использованию сервисов Яндекс.Облака за выбранный период.

## Посмотреть детализацию использования  {#show-detalization}

По умолчанию загружаются данные на текущий день с начала месяца. Данные могут обновляться с небольшой задержкой (до нескольких часов). 

Чтобы посмотреть детализацию за произвольный период, воспользуйтесь фильтрами:

1. Задайте период в календаре.
1. Выберите сервис.
   <br/>Результаты отобразятся в таблице:

Название | Описание
----- | -----
Период | Выбранный промежуток времени.
Сервис | Название сервиса, в рамках которого потреблялись ресурсы.
Продукт | Название продукта.
Ед. измерения | Единица измерения продукта.
Объем потребления | Объем потребленных ресурсов.
Сумма | Сумма, соответствующая объему потребленных ресурсов. В рублях, с учетом НДС.
Итоговая сумма | Итоговая сумма, соответствующая объему потребленных ресурсов, по всем строчка детализации. В рублях, с учетом НДС.

## Выгрузка детализации в файл {#csv-detalization}

Если вам нужно посмотреть всю историю использования сервисов или иначе отсортировать содержимое, можно выгрузить данные в файл в формате CSV (открывается в Excel и других редакторах для просмотра таблиц). Для этого [задайте фильтры](#show-detalization) и нажмите кнопку **Загрузить CSV**.
