# Изменить способ оплаты

Изменить способ оплаты можно только в платежном аккаунте c типом **Бизнес-аккаунт**: вместо оплаты банковской картой вы можете использовать только перевод с расчетного счета. 

Для этого отправьте запрос на [электронную почту](mailto:cloud_docs@support.yandex.ru), указав полное название организации и идентификатор облака (cloud id).

К запросу также прикрепите документы: 
  - Копию свидетельства о регистрации юридического лица.
  - Копию решения/протокола о назначении генерального директора (если вы являетесь генеральным директором).
  - Копию доверенности представителя юридического лица (если вы являетесь представителем).

> [!NOTE]
>
> После того, как ваша карта будет отвязана от платежного аккаунта, [пополнять лицевой счет](../operations/pay-the-bill.md) можно будет только с помощью [перевода средств с расчетного счета (через банк)](../payment/payment-methods-business.md).
>
