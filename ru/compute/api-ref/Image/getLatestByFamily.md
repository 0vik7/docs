# Метод getLatestByFamily
Возвращает последний образ из указанного семейства образов.
 

 
## HTTP-запрос
`GET /compute/v1/images:latestByFamily`
 
## Query-параметры {#query_params}
 
Name | Description
--- | ---
folderId | Обязательное поле. Идентификатор каталога для получения образа. Чтобы получить идентификатор каталога, используйте запрос [list](/docs/resource-manager/api-ref/Folder/list).  Максимальная длина — 50 символов.
family | Имя семейства образов для поиска.  Значение должно быть длиной от 3 до 63 символов и соответствовать регулярному выражению `` ^[a-z][-a-z0-9]{1,61}[a-z0-9]$ ``.
 
## Ответ {#responses}
**HTTP Code: 200 - OK**

Ресурс Image
