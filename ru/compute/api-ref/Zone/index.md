# Zone
Набор методов для получения информации о зонах доступности.
## JSON-представление {#representation}
```json 
 {
  "id": "string",
  "regionId": "string",
  "status": "string"
}
```

## Методы {#methods}
Метод | Описание
--- | ---
[get](get.md) | Возвращает информацию об указанной зоне доступности.
[list](list.md) | Возвращает список зон доступности.
