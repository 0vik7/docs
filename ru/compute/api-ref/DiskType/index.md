# DiskType
Набор методов для получения информации о типах дисков.

## JSON-представление {#representation}
```json 
 {
  "id": "string",
  "description": "string",
  "zoneIds": [
    "string"
  ]
}
```

## Методы {#methods}
Метод | Описание
--- | ---
[get](get.md) | Возвращает информацию об указанном типе диска.
[list](list.md) | Возвращает список доступных типов дисков.
