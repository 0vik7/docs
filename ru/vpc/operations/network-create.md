# Создать облачную сеть

Чтобы создать [облачную сеть](../concepts/network.md): 
1. Откройте раздел **Virtual Private Cloud** в каталоге, где требуется создать облачную сеть.
1. Нажмите кнопку **Создать сеть.**
1. Задайте имя сети. Имя может содержать строчные буквы латинского алфавита, цифры и дефисы. Первый символ должен быть буквой. Последний символ не должен быть дефисом. Максимальная длина имени — 63 символа.
1. Нажмите кнопку **Создать сеть**.

После этого будет создана облачная сеть.
