# Сервисные аккаунты

_Сервисный аккаунт_ — учетная запись, от имени которой написанная вами программа может выполнять действия в Яндекс.Облаке. Сервисный аккаунт может управлять другими ресурсами через API.

Сервисный аккаунт при этом является ресурсом: как и виртуальная машина, сервисный аккаунт создается внутри каталога. Вы можете выдавать и отнимать права доступа на сервисный аккаунт другим пользователям.

На данный момент сервисные аккаунты используются для доступа к ресурсам сервиса [!KEYREF objstorage-full-name]. Для этого необходимо создать ключи доступа. Эти ключи используются для управления ресурсами [!KEYREF objstorage-name] через API, совместимый с Amazon S3. Подробнее читайте в разделе [[!TITLE]](../../../storage/s3/index.md) документации сервиса [!KEYREF objstorage-name].
