# Добавление пользователя в облако

Эта инструкция описывает, как добавить в облако пользователя с аккаунтом на Яндексе. Если вы хотите добавить пользователя из SAML федерации, воспользуйтесь другой инструкцией.

{% list tabs %}

- Консоль управления

  1. {% include [grant-role-console-first-steps](../../../_includes/iam/grant-role-console-first-steps.md) %}
  2. На странице **Пользователи и роли** в правом верхнем углу нажмите **Добавить пользователя**.
  3. Введите электронную почту пользователя в Яндексе.
  4. Нажмите кнопку **Добавить**.

{% endlist %}
