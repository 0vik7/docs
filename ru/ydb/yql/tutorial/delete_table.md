# Удаление таблицы

Удалите [созданные](create_demo_tables.md) таблицы с помощью конструкции [DROP TABLE](../reference/syntax/drop_table.md).

```sql
DROP TABLE episodes;
DROP TABLE seasons;
DROP TABLE series;
```
