# Правила тарификации для {{ lockbox-name }}

Расчет стоимости использования {{ lockbox-name }} складывается из:

* Количества секретов.
* Числа выполненных операций [get](api-ref/Payload/get) с секретом.



{% note warning %}

Цены действуют с 5 апреля 2023 года.
